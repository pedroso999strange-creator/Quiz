# Quiz - Equipe Híbrida

Este projeto é um quiz interativo desenvolvido como ferramenta de diagnóstico para gestores de equipes de revenda. O objetivo é aumentar a consciência sobre problemas na gestão e preparar o usuário para uma solução (Mini-VSL).

## Estrutura do Projeto

- **index.html**: Estrutura principal da página e do quiz.
- **style.css**: Estilos visuais e animações (CSS).
- **script.js**: Lógica do quiz, pontuação e interatividade.

## Como Usar

1. Abra o arquivo `index.html` em seu navegador para testar o quiz localmente.
2. O quiz guiará o usuário através de perguntas de diagnóstico (SPIN Selling).
3. Ao final, um resultado será apresentado com uma chamada para ação (CTA).

## Instalação

Não há instalação necessária. Apenas baixe os arquivos e abra-os em um navegador web moderno.
